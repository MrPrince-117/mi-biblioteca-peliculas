# mi-biblioteca-peliculas
Aplicación Android desarrollada en Java para la asignatura de Programación Multimedia.

## Funcionalidades
- Registro e inicio de sesión
- Gestión de películas (CRUD)
- Filtro por género y ordenación
- Selección de portadas
- Estadísticas
- Base de datos SQLite con SQLiteOpenHelper

## Tecnologías
- Android Studio
- Java
- SQLite
